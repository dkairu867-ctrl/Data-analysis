# Q&T Group Sales Analysis

##  Overview

Interactive Excel dashboard analyzing Q&T Group sales performance across multiple dimensions including sales representatives, product categories, customer tiers, and regional performance.

##  Files Included

-  **Q&T Groups Dashboard.xlsx** - Interactive Excel dashboard with visualizations
-  **Q&T DATA.xlsx** - Raw sales data with customer and order information
-  **Q&T Group sales Dashboard.png** - Dashboard preview screenshot

##  Dashboard Features

### Visualizations:
- **V&T Groups Sales** - Line chart showing sales performance by representative
- **Sales Per Country Vs Product Category** - Clustered bar chart (Electronics, Furniture, Office Supplies)
- **Total Orders Per Country With Categories** - Bar chart with filters
- **Profit Margin per Representative** - Horizontal bar chart ranking sales reps
- **Sum Of Sales Per Tier By Region** - 3D pie chart (Bronze, Silver, Gold tiers)

### Key Metrics:
- Sales performance by representative
- Product category analysis
- Customer tier distribution
- Regional performance (North, South, East, West)
- Country comparison (USA vs Canada)
- Profit margin analysis

##  Tools Used

- **Microsoft Excel** - Dashboard creation and data analysis
- **Pivot Tables & Charts** - Data visualization
- **Slicers** - Interactive filtering
